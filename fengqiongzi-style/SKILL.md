# 风茕子写作风格 Skill

基于对风茕子12篇原创文章的结构化分析，提炼出可复用的写作方法论。

---

## 一、标题公式

核心原则：标题即钩子。必须有性暗示、金钱冲突、身份错位、或反常识信息中的至少一个。

### 标题类型

| 类型 | 公式 | 示例 |
|------|------|------|
| 性暗示 | 主语 + 带有暧昧/情色暗示的动作 | 他想在车上干点坏事。 |
| 身份标签冲突 | 身份标签 + 极端结果 | 40岁的她初中学历，长相一般，靠做小三搞到一套房。 |
| 反常识 | 反常理判断句 | 大婆没有早点去和小三谈心，损失了一个亿。 |
| 浓缩人生悖论 | 主语 + 最 + 对比结果 | 他乱搞一生，最终财产留给最丑的女人。 |
| 身份错位 | 伪装身份 + 时间限定 | 当个领导只一夜。 |
| 关系异化 | A关系 + 变成 + B关系 | 当夫妻关系变成通奸关系。 |
| 直击痛点概念 | 敏感词 + 陈述 | 我的开放式婚姻。 |
| 悬念式 | 未完成句/省略 | 那个鸭鸭。 |
| 一句话情节 | 完整陈述故事核心 | 丈夫说妓女最会玩花样。 |
| 情绪词 + 留白 | 情绪标签 + 句号 | 世态炎凉。 |
| **对比暴击** | A的极端 + 对比 + B的缺失 | 弟弟的儿子上私立，他的女儿没看过海。 |

### 标题避雷

- 不要写成新闻摘要。X 他给老家转了十年钱，离婚时账户余额为零。（像财经新闻标题）
- 不要写成哲理短文标题。X 你以为的支柱，其实你是工具。
- 好的标题是：一个具体事实 + 一个残酷对比，让人一秒看懂、想点进去。

### 标题规则
- 字数：8-25字
- 句号结尾：几乎每篇标题都以。（句号）结尾
- 必有核心冲突信号
- 口语化：像朋友聊天爆料，不用书面语

## 二、结构模板

标准结构：数字分章 + 6-10个小节 + 结尾金句

### 节奏分配

```
第1节（1-3段）：直接切入人物处境，不铺垫背景
  例：老周想把自己那台老式宝马换掉。
  例：徐瑛和老谷搞上，用她自己的话说，一开始不是图钱，是图他对我好。

第2节（2-4段）：引入第二个关键人物 + 关系本质
  快速交代两人关系的实质（利益？感情？搭伙？）

第3-5节：冲突逐层升级
  分歧 → 争执 → 摊牌，每小节300-500字

第6-8节：转折/意外事件/真相暴露
  核心技巧：不是和解，而是意外发现
  例：偷听到对话、发现谎言、灾难、反转

第9-10节：收尾
  不写说教总结。用人物行动或内心独白收束
  最后一段必须是金句级感悟
```

### 篇幅控制
- 总字数：2000-4000字
- 每小节：300-600字
- 对话占比：30-40%
- 内心独白占比：15-20%

## 三、人物设定规律

### 核心参数

| 维度 | 规律 |
|------|------|
| 年龄 | 30-50岁中年为主 |
| 阶层 | 中下层到中产：小老板、白领、蓝领、拆迁户 |
| 婚姻状态 | 离婚、再婚、出轨中、开放式。极少写头婚甜蜜 |
| 经济状况 | 总是刚好不够——有房但房贷重，有收入但不稳定 |
| 道德光谱 | 无纯好人或纯坏人。所有人都有可理解的私心 |

### 人物塑造铁律

1. 不写完美受害者：被出轨的女人也算计钱；被骗的小三也有贪婪
2. 不写纯粹恶人：渣男也真心付出过；小三也有令人心疼的背景
3. 每个行为必须自洽：读者能理解为什么TA会这么做
4. 阶层是第一驱动力：90%的冲突根源是钱和阶层差距

### 人物命名
- 极其朴素：老周、苏文、杨亮、于芬、徐瑛、陆泽、小英
- 不用诗意化名字，用街头随处可见的真名
- 偶尔用外号增加辨识度：大虎、卞总

## 四、开头技法

### 三种经典开头

**类型A：欲望/行动直入**
```
老周想把自己那台老式宝马换掉。
陆泽从小就被大虎欺负，大学毕业后进同一系统，还是被欺负。
```
→ 一句话：谁 + 想要什么/在做什么

**类型B：身份标签定义**
```
胡茜30出头，是个坚定的不婚不育者，此生崇尚自由。
我家庭条件比较好，父亲是知名医院的副院长，母亲是大学教授。
```
→ 先用身份标签定义人物，再揭示矛盾

**类型C：情境切入**
```
杨亮在开车时无意中看到了钟亚菲，她还是那么高挑、靓丽。
十一年前，房地产还是个热火朝天的行业。于芬听说村里的王洪光混得老厉害了。
```
→ 从具体场景或时代背景切入

### 开头铁律
- 第一句不超过30字
- 3句内交代核心冲突种子
- 绝不写宏大铺垫或环境描写开头
- 不写引语、名人名言、鸡汤句式开头

## 五、语言风格

### 叙事语调

核心定位：一个见过世面、不轻易评判、但什么都懂的朋友，在跟你讲她听来的故事。

### 语言层次（三层混用）

**层次1：街头口语（底层）**
- 妈的、JB、傻B、屌丝、捞女
- 整、搞、弄——动作极简
- 来一盘儿、那个——性事委婉
- 老母猪戴胸罩一套又一套——方言歇后语

**层次2：精准白描（中层）**
- 她全身都舒展成一种享受的状态。真舒服，他的体温永远是这样适手，从不咬人。
- 那一瞬间的不好，让她想起，这事不能就这么了了。
- 他的眼瞪得比平时大，但又没光，像一副死鱼眼，空洞得很，可以放进去死亡。

**层次3：文学金句（顶层）——仅限结尾**
- 掀开表象，全是人到中年的狼狈平庸。
- 他们对待人生从来没有这样冷静和中立过，如此地互相理解，深刻地同情，但是，永不可能再次携手。
- 人这一辈子，并不是说财富自由、情感自由，就能在诸多选择中拥有超乎常人的幸福。

### 叙事视角

- 第三人称全知为主（11/12篇）
- 偶尔第一人称（1/12篇：我的开放式婚姻）
- 第一人称只在口述实录/亲身经历时使用
- 作者偶尔跳出评论：风风说、姐妹们注意

### 对话风格

- 极简、碎片化、像真实对话
- 不使用大段独白说教
- 对话中带出阶层和性格差异
- 例句：老周说：我想换车。苏文说：换成啥车？电车，六座的，很宽敞，宽敞得咱俩可以在车上那个。你在车上那个过吗？
- 极速推进信息，不超过3轮对话就要进入实质

## 六、结尾技法

### 四种经典结尾

**类型A：金句收束**
```
掀开表象，全是人到中年的狼狈平庸。人到了一定阶段，要的不过是岁岁寻常。
```
→ 从具体情节提炼出普适感悟，但不空洞

**类型B：行动留白**
```
她把五万块钱寄给于富，告诉他什么叫大场合——把房子翻新，娶个不骗他的女人。
至于她的傻子哥哥呢，他很好，智障的毛病完全治好了，感谢生活。
```
→ 用后续行动暗示情感走向，不直接说

**类型C：画面定格**
```
她拉着女儿往后缩，跑出一身热汗来，多么庆幸当初没有为了十万块钱嫁给他。
直到她把步子走稳的时候，身上的汗凉了，才蓦然有点凄然。
```
→ 用一个具体画面收束，让画面承载寓意

**类型D：反转点题**
```
那一夜是真正的老总，不亏。
```
→ 短促有力的反转收尾，颠覆前文所有讲述

### 结尾铁律
- 不做说教式总结
- 不做道德判决
- 不给完满结局——生活本来不完满
- 最后一句必须让人停下来想

## 七、核心主题矩阵

### 必选主题维度（每篇包含2-3个）

| 主题维度 | 具体表现 |
|----------|----------|
| 钱 | 买车贷款、拆迁赔偿、欠款追讨、房子归属、彩礼工资 |
| 性 | 出轨、包养、一夜情、开放式婚姻、性交易 |
| 阶层 | 老板vs工人、城里vs农村、有钱vs赤贫、体制内vs体制外 |
| 背叛 | 出轨、欺骗、隐瞒病史/婚史、假意复合真算计 |
| 绝境 | 泥石流、杀人逃亡、破产清算、癌症、工亡 |
| 中年焦虑 | 身体衰老、收入下降、婚姻倦怠、子女教育、父母养老 |

### 永不触碰的主题
- 纯甜宠爱情故事
- 霸道总裁爱上我
- 完美主角成长记
- 单一价值观说教

## 八、性描写规则

- 直接但不色情：做爱、睡觉、来一盘儿、那个
- 性从来不是浪漫的——是交易、机械、绝望、或分手的仪式
- 用身体反应代替心理描写
- 例句：她顺从了。她跟他来到后座，脱掉了一条裤腿。他们的结合有点像某种仪式，没有缠绵的爱意，没有炽热的贪念，只是两个即将陌路的人，用最原始的相拥，为这几年的相伴做一场潦草的落幕。
- 性描写必须服务于情节推进或人物揭示，绝不为刺激而写

## 九、写作工作流（强制流程）

### 阶段一：梗概（必做，不可跳过）

**直接写全文 = 高产废稿。必须先出梗概，用户确认后再展开。**

梗概包含4个要素：

1. **一句话故事**：谁 + 核心冲突 + 结局
2. **核心戏剧事件**：具体是什么引爆/转折/反转（不能是"感情慢慢淡了"）
3. **情节节点（6-10个）**：每个节点一句话，标注核心冲突推进
4. **金句植入点**：在哪些节点埋伏金句，各是什么

### 阶段二：用户确认梗概

- 检查核心冲突是否有"核"（不是流水账）
- 检查金句密度（每2-3节至少一句）
- 检查人物道德灰度（不能纯好/纯坏）

### 阶段三：展开写作

按确认的梗概逐节展开，每节300-600字。对话占比30-40%。

### 阶段四：自检

1. 标题有钩子吗？（对比暴击、性暗示、反常识）
2. 开头3句内有冲突种子吗？
3. 每2-3节有一句金句吗？
4. 结尾让人想停下来吗？
5. 对话像真人说话吗？
6. 2000-4000字范围内吗？
7. 有没有残留的废稿片段？（只写一个故事）

## 十、旧版写作流程（参考）

### 第一步：选题（选2-3个主题维度）
1. 从主题矩阵中选2-3个主题维度
2. 组合出核心冲突
3. 提炼一句话标题

### 第二步：人物（造2-3个关键人物）
1. 确定每个人的经济状况和阶层背景
2. 确定每个人的私心是什么
3. 确保没有纯好人或纯坏人

### 第三步：情节（画出6-10个节点）
1. 起始冲突（第1节）
2. 关系揭示（第2节）
3. 冲突升级点A（第3节）
4. 冲突升级点B（第4节）
5. 冲突升级点C（第5节）
6. 转折事件（第6节）
7. 真相暴露/意外（第7-8节）
8. 情感处理（第9节）
9. 收束金句（第10节）

### 第四步：写作
1. 用口语化第三人称叙事
2. 大量对话推进剧情
3. 关键处用身体描写替代心理分析
4. 结尾用金句或行动留白

### 第五步：检查
1. 标题有钩子吗？
2. 开头3句内有冲突吗？
3. 人物有道德灰度吗？
4. 结尾让人想停下来想吗？
5. 对话听起来像真人吗？
6. 2000-4000字范围内吗？

## 十一、避雷清单

### 绝对不要
- 写纯甜宠/霸总/玛丽苏情节
- 开头用环境描写或鸡汤引言
- 人物非黑即白道德分明
- 结尾说教或强行圆满
- 性描写为刺激而写
- 通篇文艺腔调
- 给人物起太文艺的名字
- **跳过梗概直接写全文（= 高产废稿）**
- **在同一篇中写两个故事（开头废稿未删除）**
- **标题写成新闻摘要或哲理短句**
- **用"感情慢慢淡了"做核心冲突（必须有具体引爆事件）**

### 一定要有
- 标题有性暗示或金钱冲突
- 数字分章结构
- 至少一个意外反转
- 人物有可理解的私心
- 结尾金句
- 口语化对话
- 阶层冲突底色
- **先出梗概，用户确认后再写全文**

## 十二、实战案例

### 案例：弟弟的儿子上私立，他的女儿没看过海。

**梗概阶段产出**：[drafts/弟弟的儿子上私立，他的女儿没看过海.md](../../drafts/弟弟的儿子上私立，他的女儿没看过海.md)

| 要素 | 内容 |
|------|------|
| 标题类型 | 对比暴击 |
| 核心事件 | 妻子发现丈夫十年秘密转账67.8万给老家，女儿上学换房没钱 |
| 章节 | 9节 |
| 金句密度 | 7句（每1.3节一句） |
| 主题维度 | 钱 + 阶层 + 背叛 + 中年焦虑 |

### 创作过程经验

1. **第一版失败**：直接写全文，写成流水账时间线，无核心戏剧事件 → 废稿
2. **第二版成功**：先出梗概 → 确认核心冲突（秘密转账曝光） → 标注金句位置 → 再展开
3. **标题迭代**：第一版"他给老家转了十年钱..."像财经新闻 → 改为对比暴击型标题
4. **关键教训**：如果梗概阶段找不到一个"一句话说完让人想读"的核心冲突，故事就不该开始写
